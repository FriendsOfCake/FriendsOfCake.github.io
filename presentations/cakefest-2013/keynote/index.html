New Query builder

Create queries using a fluent interface with automatic type conversion


$union = (new Query($connection))
	->select(['id', 'title'])
	->from(['a' => 'articles']);

$results = (new Query($connection))
	->select(['id', 'comment'])
	->from(['c' => 'comments'])
	->where(['created >=', new DateTime('1 day ago')])
	->union($union)
	->execute();

Smart association eager loading

This will use a single query


$table = Table::build('author');
$table->hasMany('article', [
	'property' => 'articles',
	'strategy' => 'subquery',
	'sort' => ['article.id' => 'asc']
]);
$results = $table->find('all')->contain('article')->toArray();
